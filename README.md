# YW_DEMOS
Demo programs for some library I ported to delphi, including:
- [XXHASH](https://github.com/YWtheGod/XXHASH4Delphi)
- [ZSTD](https://github.com/YWtheGod/ZSTD4Delphi)
- [LZ4]{https://github.com/YWtheGod/LZ4Delphi}
Porting C libray to Delphi in Windows need a C Runtime Library to successfully link. So, you will need this:
- [LIBC](https://github.com/YWtheGod/LIBC)

一些我移植的C功能库的DEMO程序集中放置仓库，包括：
- [XXHASH](https://gitee.com/YWtheGod/XXHASH4Delphi)
- [ZSTD](https://gitee.com/YWtheGod/ZSTD4Delphi)
- [LZ4]{https://gitee.com/YWtheGod/LZ4Delphi}
